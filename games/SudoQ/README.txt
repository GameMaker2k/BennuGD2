Bombergames Soft 2016
---------------------
SudoQ 0.4 for PC by BoMbErLiNk.
Written in BennuGD, ported from Fenix.
BennuGD2 Port by SplinterGU (2021)

FEATURES
--------
* Basic Sudoku rules
* Random board generator
* Board resolution
* 3 levels of difficulty
* Savestate 
* Different backgrounds

CONTROLS
--------
KEYBOARD arrows/numbers or MOUSE supported.
DEL - delete number
KEY A/D - Change background
ALT+ENTER for full screen
F1 - 320x240
F2 - 640x480 (scaled)
ESC - Exit & Save
